SAE Tetris

Septembre 2024-Janvier 2025

Ce projet consiste à développer un jeu Tetris en fonction des demandes clients dans le cadre de la programmation en Python

Fonctionnalités principales :

  • Reproduction d'un jeu de base avec les 5 pièces donnés.
  • Ajouts de plusieurs variantes possibles
  • Sauvegarde de jeu et de paramètres
  • Ajouts de mode bonus

Projet UNESCO

Septembre 2024-Juin 2025

Projet site web où le but est de créer un site internet, sur un lieu touristique en partenariat avec l'UNESCO , dans notre cas les quais de seine de Paris. Nous utilisons du HTML , CSS ainsi que du PHP

Le site comprend :

  • Une page d'accueil avec différents articles
  • Différents parcours d'itinéraires
  • Des pages d'histoire sur le lieu
